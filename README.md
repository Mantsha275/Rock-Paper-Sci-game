## 📌 Project Overview

This project is a browser-based implementation of the classic **Rock Paper Scissors** game.

The player selects one of the three choices:

* 🪨 Rock
* 📄 Paper
* ✂️ Scissors

The computer randomly generates its choice, compares it with the player's selection, and immediately displays the result.

The game keeps track of both the player's score and the computer's score throughout the session.

## 🎮 How to Play

1. Open the game in your browser.
2. Click on either:

   * Rock
   * Paper
   * Scissors
3. The computer randomly chooses its move.
4. The winner is decided according to the game rules.
5. Scores update automatically.
6. Play as many rounds as you like.

---

## 📖 Game Rules

* Rock beats Scissors.
* Scissors beats Paper.
* Paper beats Rock.
* Same choices result in a Draw.
